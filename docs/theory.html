Social Network Analysis for Social Scientists

Chapter 3 Theory

We already introduced egonets in section 1.4.1. We defined an egonet as the set of ties surrounding sampled individual units. (cf. Marsden 1990). Thus we have an ego with ties to one or more alters. If we also know the ties between our alters we may observe triads. In the figure below I have made the triad green.

Figure 3.1: A Triad within an egonet

Let us discuss this important network structure in some more detail first, before we move on with our discussion on Egonets.

3.1 Network Structures (TRIAD)

Let us start with all possible triad configurations if we have a (binary) undirected tie. See Figure 3.2.

Figure 3.2: undirected triad configurations

We observe an unconnected triad, a triad with a connected pair, an open triad and a closed triad. The open triad is also called a ‚Äòforbidden triad‚Äô and actor i in such a triad is said to hold a ‚Äòbrokerage position.‚Äô
